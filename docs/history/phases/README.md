# Phase Report Archive

This directory preserves delivery reports from Foundation phases 1C.2 through 1D.4A.

These files are historical evidence, not independent sources of current architecture. For current product truth read, in order:

1. `../../../PROJECT_VISION.md`;
2. `../../../AI_PROJECT_MEMORY.md`;
3. `../../../ARCHITECTURE.md`;
4. `../../../ROADMAP_NEXT.md`;
5. the accepted ADRs in `../../adr/`.

`PHASE_1D4A_REPORT.md` is the current-phase delivery report and remains mandatory reading for foundation changes, but accepted ADRs and active architecture documents take precedence if wording differs.
