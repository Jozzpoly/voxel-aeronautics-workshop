# VAW Visual Asset Pack V1 — preview/test sample

This sample uses the embedded `test_anim.gltf` regression asset.

It is intentionally marked as `preview-test-asset`. It exists to test Studio export, validation, embedded textures/buffers, node bindings, and clip bindings. It is not a final VAW gameplay asset.

Preview can load the model even if a user edits the manifest into an invalid state. Export to VAW must stay blocked until the manifest passes V1 validation.
